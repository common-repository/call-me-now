== Description ==

This is a dutch-based plugin for VoIP services. It works with companies like Verbonden B.V. Voys or other VoIP providers. If you're a VoIP provider and you're not sure if you can use this, you probably cant =)

= Nederlands =
Dit is een [Call Me Now](https://verbonden.nl/voip-modules/call-me-now) widget voor telefonie diensten van [Verbonden B.V.](https://verbonden.nl), maar kan ook ingezet worden door andere VoIP providers (zoals Voys) die deze widget ondersteunen. Ben je aangesloten bij een VoIP provider en vraag je je af of je deze widget kunt gebruiken? Informeer dan bij Verbonden of zij je hier mee kunnen helpen. Verbonden levert VoIP telefonie voor klein, midden en groot bedrijf. Met deze pugin plaats je op een super eenvoudige manier een formulier waarmee bezoekers contact op kunnen nemen, realtime. 

Wanneer men het nummer invoert in de widget en deze verzend wordt er direct (realtime) een verbinding opgelegd tussen jouw VoIP account en het nummer van de klant.

Voor meer informatie over VoIP diensten kun je kijken op de [website van Verbonden](https://verbonden.nl).

== Installation ==
1. Upload the plugin to your WordPress site. This can be done manually (through FTP) and through the WordPress dashboard (as *.zip file)
2. Activate the plugin through the plugin page
3. Add the widget through Appearance -> Widgets and drag the Call me now widget to your sidebar.
